# Keyboard Shortcuts

Open **Keyboard Shortcuts** from the toolbar to search commands, record new bindings, import or export a keybind preset, and reset all shortcuts to defaults.

## Playback

| Action | Shortcut |
|---|---|
| Play / Pause | `Space` |
| Previous / Next frame | `Left` / `Right` |
| Previous / Next snap point | `Up` / `Down` |
| Go to start / end | `Home` / `End` |

## Editing

| Action | Shortcut |
|---|---|
| Split at playhead | `Ctrl+K` / `Alt+C` |
| Split at cursor | `Shift+C` |
| Join clips | `Shift+J` |
| Delete selected | `Delete` |
| Ripple delete | `Ctrl+Delete` |
| Freeze frame | `Shift+F` |
| Nudge item, 1px / 10px | `Shift+Arrow` / `Ctrl+Shift+Arrow` |
| Group / Ungroup tracks | `Ctrl+G` / `Ctrl+Shift+G` |

## History and UI

| Action | Shortcut |
|---|---|
| Undo / Redo | `Ctrl+Z` / `Ctrl+Shift+Z` |
| Toggle snap | `S` |
| Toggle keyframe editor | `Ctrl+Shift+A` |
| Open Scene Browser | `Ctrl+Shift+F` |
| Zoom in / out | `Ctrl+=` / `Ctrl+-` |
| Zoom to fit | `\` |
| Zoom to 100% | `Shift+\` |
