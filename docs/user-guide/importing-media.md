# Importing Media

Import media into the project library before placing it on the timeline.

## Import files

Use **Import** or **Import media** in the Media panel, then choose files from disk. You can also drag and drop files into the media library.

Folders are not supported yet. Drop media files directly.

## Import from URL

Use **Import Media From URL** for a direct URL to a video, audio, image, or GIF file. The URL must point to the media file itself, not a page that embeds it.

## Supported media categories

FreeCut organizes imported assets as video, audio, image, GIF, SVG, generated assets, and compound clips. Codec support depends on what the browser can decode.

## Media cards and info

Media cards expose menus for **File**, **Proxy**, **Transcript**, **Embedded captions**, and **Menu AI**. Use **Media info** to inspect codec, dimensions, duration, frame rate, size, type, and transcript state.

## Source Monitor

Use **Open In Source Monitor** from media info to preview a source asset before editing it into the timeline. The source monitor supports in/out points plus insert and overwrite edits.

## Proxies, transcripts, and captions

Use **Generate Proxy** for easier editing with heavy video files. Use **Generate Transcript** or **Refresh Transcript** for speech text. Use **Analyze with AI** where available for scene and caption workflows.

Use **Extract Embedded Subtitles** or the **Embedded subtitles** dialog when a file includes subtitle tracks. Extracted subtitles can be inserted into the timeline.

## Missing Media

If FreeCut shows **Missing Media**, the linked source file needs permission or is no longer at the recorded path. Use **Grant Access**, **Locate**, **Locate Folder**, or **Browse Another Folder**. Choose **Work Offline** only when you want to keep editing before relinking.
