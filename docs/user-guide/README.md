# FreeCut User Guide

This guide is for using FreeCut. For setup, architecture, and contribution notes, use the developer documentation in the repository root.

FreeCut is a local-first, browser-based editor. Projects, media references, generated assets, caches, and exports stay in a workspace folder you choose; editing and export run in the browser.
