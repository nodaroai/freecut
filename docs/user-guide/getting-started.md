# Getting Started

Use this path for a first edit from launch to export.

## Browser

Use Chrome or Edge 113+ for the full workflow. FreeCut depends on WebGPU, WebCodecs, OPFS, and the File System Access API.

Brave can work, but its File System Access API may be disabled. Open `brave://flags/#file-system-access-api`, set the flag to **Enabled**, then relaunch Brave.

## First edit

1. Open FreeCut in a supported browser.
2. When FreeCut asks you to pick a workspace folder, choose a normal folder you can read and write. This is where projects, caches, generated assets, and exports live.
3. On the Projects page, choose **New Project** or **Create Project**.
4. In the Media panel, use **Import** or **Import media**. You can also drag media files into the media library.
5. Drag imported media to the timeline.
6. Press `Space` to preview playback. Use `Left` and `Right` for frame stepping.
7. Use **Save** or `Ctrl+S` as you work.
8. Choose **Export** or **Export Video**, check the export settings, then export from the browser.

FreeCut keeps source media local. Local files are linked from your workspace; they are not uploaded to a server.
