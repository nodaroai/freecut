# Editor Layout

FreeCut uses dense panels and direct controls. The main surfaces are stable across projects.

## Projects page

The Projects page is where you create, open, duplicate, import, export, and delete projects. It also shows the active workspace folder.

## Editor toolbar

The Editor toolbar contains **Back to Projects**, workspace tabs, project specs, **Settings**, **Keyboard Shortcuts**, **Save**, and **Export**.

## Edit workspace

The **Edit** workspace is the default cutting layout. Use it for media organization, timeline edits, text, shapes, effects, transitions, AI tools, preview playback, and clip properties.

## Color workspace

The **Color** workspace focuses on grading. Select a clip, then use the Color panel and scopes when they are visible.

## Media sidebar

The sidebar tabs are **Media**, **Text**, **Shapes**, **Effects**, **Transitions**, and **AI**.

- **Media** holds imported assets, compound clips, media info, proxies, transcripts, captions, and missing media controls.
- **Text** and **Shapes** create generated timeline items.
- **Effects** and **Transitions** add clip processing and edits between clips.
- **AI** contains local browser tools such as text to speech and music generation when supported.

## Preview monitor

The preview monitor shows the current frame, canvas transforms, overlays, masks, scopes, and playback state.

## Timeline

The **Timeline** contains video and audio tracks, clip items, markers, in/out points, snapping, trim tools, and zoom controls.

## Properties panel

The **Properties** panel edits selected clips. Depending on the selection, it can show Video, Audio, Effects, Transform, Text, Subtitle, Shape, Caption, and Color controls.

## Keyframe editor

The **Keyframe editor** edits animation curves and timing for selected keyframed properties. It supports graph and dopesheet-style views.

## Settings and shortcuts

Use **Settings** for general, timeline, AI, and storage preferences. Use **Keyboard Shortcuts** to search commands, record bindings, import or export keybind presets, and reset defaults.

## Editor terms

- Workspace: a folder FreeCut can read and write.
- Project: the timeline, settings, and media references for one edit.
- Media: linked video, audio, image, SVG, GIF, generated, or compound assets.
- Timeline: the track-based edit area.
- Source monitor: the media preview entry point for in/out and insert or overwrite edits.
- Export: a rendered video or audio file written by the browser.
